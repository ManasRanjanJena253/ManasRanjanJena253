## Summary

I build reproducible, end-to-end machine learning systems that bridge research and production. My work spans generative models, reinforcement learning agents, full-stack ML tooling, applied NLP/LLM systems, and high-performance Go backends. I emphasize clean engineering, measurable impact, and deployable systems rather than experiments that stop at a notebook.

- **Current:** B.Tech (Mathematics & Computing) — Punjab Engineering College, Chandigarh (Class of 2028)
- **Started focused ML/AI work:** September 2024
- **Primary focus areas:** LLM / LCM engineering, generative AI systems, Go backend systems, and robust model deployment

## How I Work

- **Reproducibility first** — deterministic seeds, versioned datasets, experiment tracking.

- **Explainability & meaningful metrics** — optimize for robust improvements tied to measurable goals.
